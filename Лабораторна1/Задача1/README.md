# Лабораторна робота №1

## Завдання 1
[cite_start]Написати програму, яка виконує арифметичні дії з введеними числами, обчислює параметри прямокутника (з приведенням типів) та аналізує виділену пам'ять[cite: 24, 28].

## Опис роботи коду
1. Програма запитує два цілих числа.
2. Виводить суму, різницю, добуток та частку.
3. Запитує довжину та ширину прямокутника (цілі числа).
4. [cite_start]Перетворює їх у дійсні числа (double)[cite: 27].
5. Обчислює площу та периметр.
6. [cite_start]Виводить адреси пам'яті змінних та загальний обсяг використаної пам'яті[cite: 28].

## Використані елементи С++
* `cin`, `cout`: Ввід та вивід даних.
* `int`, `double`: Типи даних.
* `(double)variable`: Явне приведення типів.
* [cite_start]`&variable`: Оператор взяття адреси[cite: 4, 41].
* [cite_start]`sizeof(variable)`: Оператор визначення розміру пам'яті[cite: 4, 42].

## Інструкція
1. Запустіть програму.
2. Введіть два числа (наприклад, `10` та `5`).
3. Введіть розміри прямокутника (наприклад, `4` та `8`).
4. Перегляньте результати обчислень та аналіз пам'яті.
